# TechnoForged - Modern Software Development Agency Website

A sleek, modern landing page for TechnoForged, a software development company specializing in creating exceptional digital experiences that drive business growth.

![TechnoForged](https://img.shields.io/badge/TechnoForged-Software%20Development-00ff88)
![License](https://img.shields.io/badge/license-MIT-blue.svg)
![Version](https://img.shields.io/badge/version-1.0.0-green.svg)

## 🚀 Features

- **Modern Design**: Clean, professional interface with glassmorphism effects
- **Fully Responsive**: Optimized for all devices from mobile to desktop
- **Smooth Animations**: Engaging scroll-triggered animations and micro-interactions
- **Performance Optimized**: Fast loading times with optimized assets
- **SEO Ready**: Semantic HTML structure for better search engine visibility
- **Accessibility**: ARIA labels and keyboard navigation support

## 🛠️ Technologies Used

- **HTML5**: Semantic markup
- **CSS3**: Custom properties, Grid, Flexbox, animations
- **Vanilla JavaScript**: No framework dependencies for optimal performance
- **Google Fonts**: Outfit font family
- **SVG Icons**: Scalable vector graphics for crisp icons

## 🚀 Quick Start

1. **Clone the repository**
   ```bash
   git clone https://github.com/fen0dev/TechnoForged.git
   cd TechnoForged
   ```

2. **Open the website**
   - Simply open `html/index.html` in your web browser
   - Or use a local server:
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js
   npx serve
   ```

3. **View the website**
   - Navigate to `http://localhost:8000/html/index.html`

## 🎨 Customization

### Colors
Edit the CSS custom properties in `css/style.css`:
```css
:root {
    --bg-black: #0b0b0b;
    --accent-primary: #007c42;
    --accent-secondary: #7b3ff2;
    --accent-tertiary: #ff006e;
}
```

### Content
- Update company information in `index.html`
- Modify service cards, portfolio items, and process steps
- Replace placeholder social media links with actual URLs

### Typography
The website uses the Outfit font family. To change it, update the Google Fonts link in `index.html`.
